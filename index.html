Pertanyaan yang Sering Ditanyakan

Belum menemukan jawaban? Langsung tanya via WhatsApp — gratis.

Tergantung kompleksitas sistem. Untuk sistem sederhana (form + dashboard) biasanya 3–7 hari kerja. Sistem menengah (multi-user, approval, PDF) 1–3 minggu. Sistem kompleks (multi-modul, integrasi banyak layanan) 3–6 minggu. Estimasi tepat diberikan setelah konsultasi dan analisis kebutuhan.
Untuk sistem berbasis Google Apps Script + Google Sheets, tidak ada biaya server bulanan. Sistem berjalan di atas Google Workspace yang sudah Anda miliki. Jika membutuhkan hosting eksternal (misal WordPress), biaya hosting dan domain ditanggung klien (estimasi Rp 150rb–300rb/tahun).
Revisi minor (perubahan teks, warna, urutan kolom, dsb.) tidak dibatasi selama masa pengerjaan. Revisi mayor (perubahan alur bisnis atau fitur baru di luar scope) akan dibahas dan disesuaikan biayanya. Saya selalu memastikan sistem benar-benar sesuai kebutuhan sebelum serah terima.
Tidak perlu. Sistem dirancang agar mudah digunakan oleh non-teknisi. Setelah selesai, saya memberikan training penggunaan lengkap dan panduan tertulis (SOP). Jika ada kendala penggunaan setelah serah terima, bisa langsung menghubungi via WhatsApp.
Data tersimpan di Google Sheets / Google Drive milik Anda sendiri — bukan di server saya. Saya tidak memiliki akses permanen setelah serah terima kecuali diizinkan untuk keperluan maintenance. Keamanan data sepenuhnya dalam kontrol akun Google Workspace Anda.
